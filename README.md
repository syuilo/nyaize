nyaize

[![][npm-badge]][npm-link]
[![][mit-badge]][mit]

Nyaization a text

## Install
``` shell
$ npm install nyaize
```

## Usage
``` javascript
nyaize(text)
```

## Example
``` javascript
import nyaize from 'nyaize';

nyaize('ひなビタはいいぞ'); // => 'ひにゃビタはいいぞ'
```
